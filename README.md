# EduPath AI

EduPath AI is a premium, modern, AI-powered Unified Student Engagement Platform for Indian students planning higher education abroad.

## Tech Stack
- Frontend: Vanilla HTML/CSS/JS (no framework, no npm)
- Backend: Python FastAPI
- Database: SQLite
- AI: Anthropic Claude (claude-sonnet-4-20250514)

## Setup & Running Locally

1. **Navigate to backend:**
```bash
cd backend
```

2. **Install dependencies:**
```bash
pip install -r requirements.txt
```

3. **Set up environment variables:**
```bash
cp .env.example .env
# Open .env and add your Claude API key
```

4. **Run the backend server:**
```bash
python main.py
```

5. **Open app:**
Navigate to [http://localhost:8000](http://localhost:8000)

## Features
- AI-Powered Chatbot Advisor mentor
- Career Navigator and Recommendation Engine
- ROI Calculator
- Visa and Application predictor
- Education Loan Estimator
